## LPS – Lern- & Prüf-Simulator
Browserbasierte Angular-Anwendung zur Vorbereitung auf LPIC-1-Zertifizierungsprüfungen.

## Modi
Lernmodus – Richtige Antworten werden sofort angezeigt
Teil-Prüfmodus – Direktes Feedback nach jeder Antwort
Voll-Prüfmodus – 60 zufällige Fragen, Ergebnis am Ende (bestanden bei max. 8 Fehlern)

## Fragenkataloge
LPIC-101 – Linux Administrator Teil 1 (120 Fragen)
LPIC-102 – Linux Administrator Teil 2 (120 Fragen)

## Installation
bashnpm install
npm start
Anwendung läuft unter: http://localhost:4200
Weitere Befehle
bashnpm run build   # Produktions-Build
npm test        # Unit-Tests ausführen
